WegTool : 
	It is a netbean project. It contain all source files in src folder and executable jar file in dist folder.
	You can directly open this project from netbean and start working on it.
	
	For creating executable jar file : select project and then select "clean and build project" option.
	It will make wegtool.jar file in dist folder.

WegJar :
	It contain executable jar file . You can directly run the project using this jar file
